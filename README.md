# Natural Language Processing (NLP) Experiments

A comprehensive collection of **Natural Language Processing (NLP)** experiments implemented using **Python**, **NLTK**, **spaCy**, **Scikit-learn**, **Gensim**, and **Transformers**. This repository is intended for undergraduate and postgraduate students, educators, and researchers to understand fundamental as well as advanced NLP concepts through hands-on implementations.

---

## 📚 Course Information

**Course:** Natural Language Processing (NLP)

**Department:** Artificial Intelligence and Data Science

**Instructor:** Vijay Shanker

---

## 🎯 Objectives

This repository aims to:

- Understand the fundamentals of Natural Language Processing.
- Perform text preprocessing and feature extraction.
- Implement syntactic and semantic analysis techniques.
- Build machine learning models for NLP tasks.
- Explore modern transformer-based language models.
- Apply NLP techniques to real-world applications.

## 🛠️ Software Requirements

- Python 3.10+
- Jupyter Notebook / Google Colab
- Visual Studio Code (Optional)

### Required Python Libraries

```bash
pip install nltk
pip install spacy
pip install scikit-learn
pip install pandas
pip install numpy
pip install matplotlib
pip install seaborn
pip install gensim
pip install wordcloud
pip install textblob
pip install transformers
pip install datasets
pip install sentencepiece
pip install torch
```

Download NLTK resources:

```python
import nltk

nltk.download('punkt')
nltk.download('stopwords')
nltk.download('wordnet')
nltk.download('averaged_perceptron_tagger')
nltk.download('omw-1.4')
```

Download spaCy English model:

```bash
python -m spacy download en_core_web_sm
```
